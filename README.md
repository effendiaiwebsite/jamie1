# DORC Enterprises Website

A modern, single-page website for DORC Enterprises, a technical recruitment company run by Jamie Couper.

## Features

- **Single-page design** with smooth scrolling navigation
- **Responsive layout** that works on all devices
- **Modern UI** with clean, professional styling
- **Contact information** prominently displayed
- **Social media links** section (ready for integration)
- **Service offerings** clearly presented

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Interactive features and smooth scrolling

## Setup

Simply open `index.html` in a web browser to view the website. No build process or dependencies required.

## Customization

To add social media links, update the `href` attributes in the social links section of `index.html`:

```html
<a href="YOUR_LINKEDIN_URL" class="social-link" aria-label="LinkedIn">
<a href="YOUR_TWITTER_URL" class="social-link" aria-label="Twitter">
<a href="YOUR_GITHUB_URL" class="social-link" aria-label="GitHub">
```
